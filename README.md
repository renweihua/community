#### 软件架构
软件架构说明（各自文件下有对应的安装流程介绍）
1. laravel APi
	* 编程语言：`PHP7.3+`
	* 后端框架： `Laravel8`
	* 前端Vue框架：`vue-element-admin`
	* Nodejs  v14.*
2. hyperf socket服务端
3. uni-app 汉服荟社区
